# Swift Learnings

My personal Swift learning journey. This repo tracks my progress as I explore the language, its standard library, and the broader Apple ecosystem (Foundation, SwiftUI, etc.).

## Structure

Each topic or experiment lives in its own file or directory. Files are intentionally small and self-contained so they can be read top-to-bottom as study notes.

## Running

Run a single file with the Swift toolchain:

```sh
swift <file>.swift
```

Or open the file in Xcode / VS Code with the Swift extension and run it from there.
